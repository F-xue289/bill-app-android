# bill-app-android

# 记账APP（Consumption Record APP）

## 项目简介
这是一款基于Android的简易记账应用，帮助用户记录日常收支，实现数据的本地持久化存储与管理。

## 技术栈
- 语言：Java
- 框架：Android Jetpack（ViewModel、LiveData、Room）
- 数据库：Room
- 版本控制：Git + GitHub

## 功能列表
1.  首页显示总收支金额
2.  新增/删除账单记录
3.  账单列表展示
4.  数据本地持久化存储

## 项目运行方式
1.  克隆项目到本地：`git clone https://github.com/F-xue289/bill-app-android.git`
2.  使用Android Studio打开项目
3.  连接模拟器或真机，运行APP即可

## 开发说明
项目按教学周迭代开发，从基础界面到完整数据库功能，全程使用Git进行版本管理，提交记录清晰可查。
